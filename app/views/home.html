Single Page Web Apps

Single page web apps offer a more 'native-app-like' experience to the user. Created with standard web technologies - HTML, CSS and Javascript these sites are generally much faster than traditional sites as they only reload the section(s) of data that change each time the user interacts with the site instead of reloading the entire page.
