Lovegrove Design - Job Manager

LOGIN - secure login page using Laravel mcrypt PHP extension.

LIVE JOBS - displays a monthly list of active client jobs with total time.

CLIENT LIST - full list of client database with dynamic 'display as you type' search results.

ADD CLIENT - add new client details and assign agreed hourly rate.

WORK VOLUME GRAPHS - displays graphical interface of monthly work volumes for each client over a specified year.

ADD JOB DETAILS - quickly add completed tasks to client jobs recording description and time spent.

COMPARE HISTORY - quickly compare and review monthly totals over the same period in previous years.

Scope:

To create an online job / project management tool. The requirements were to centralise all client contact details, have the abiity to create jobs, track time and details against each job, analyse client monthly and yearly job histories and revenues.

Responsive:

Full responsive design, not essential as the primary focus was a desktop application but useful when needing to access client data in out of the office situations.

Technologies used:

HTML5, CSS3 (Bootstrap), AngularJS, PHP (Laravel framework), MySql database.
